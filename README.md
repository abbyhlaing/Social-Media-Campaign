# Social Media Campaign Website

## Overview
The Social Media Campaign Website is a full-stack web application designed to manage and display social media campaign content through separate user and admin interfaces. The platform allows administrators to manage campaign information dynamically while users can browse content and submit contact forms.

The system includes authentication, role-based page access, and a responsive interface built for content management and user engagement.

---

## Features

### User Features
- Browse campaign information
- View dynamic content from admin dashboard
- Submit contact forms
- Responsive user interface
- Separate pages for registered and guest users

### Admin Features
- Admin dashboard
- Add, edit, update, and delete campaign data
- Manage website content dynamically
- View and manage user-submitted information
- Authentication and role management

## Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository-name.git
cd your-repository-name
```

Install and configure your local server environment:
- XAMPP / WAMP / MAMP recommended

Move the project folder into:

```bash
htdocs/
```

---

## Run the Project

1. Start Apache and MySQL
2. Import the database into phpMyAdmin
3. Open your browser and visit:

```bash
http://localhost/project-folder-name
```
